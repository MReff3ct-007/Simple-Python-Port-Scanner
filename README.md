# Simple Python Port Scanner

This is a lightweight and user-friendly port scanner written in **Python 3**.

## Description
The script allows users to scan an IP address or website for open TCP ports and displays the detected services in a clear terminal interface.  

## Made With
- **Python 3**

## Modules Used
- `socket` – Network connections and port scanning  
- `sys` – Script control and exits  
- `re` – Input validation (IP/hostname)  
- `datetime` – Scan timestamps  

## Usage
Run the scanner using:
```bash
python3 run.py
